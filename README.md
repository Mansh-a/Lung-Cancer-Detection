  <h2>📁 Repository Structure</h2>
  <pre>
.
├── Lung_cancer(SVC).ipynb                     # SVC model implementation

  </ul>

  <h3>Features:</h3>
  <ul>
    <li>Data preprocessing</li>
    <li>Model training and evaluation</li>
    <li>Performance analysis</li>
    <li>Visual insights</li>
  </ul>
